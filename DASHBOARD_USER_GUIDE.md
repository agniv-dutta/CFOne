# Financial Intelligence Dashboard - Quick Guide

## Accessing the Dashboard

1. **Upload & Analyze** a financial document (PDF or Excel)
2. **Wait for completion** - Status will show "completed"
3. **Click "📊 Dashboard" button** on the Report page
4. **View interactive charts** powered by Amazon Nova AI

## Dashboard Features

### Overview Tab
Displays 7 interactive financial charts:

#### 1. **Revenue Waterfall** (Top Left)
- Shows how revenue breaks down into components
- Green = positive contributors (new customers, expansion)
- Red = headwinds (churn, discounts)
- **Insight**: AI explains revenue drivers

#### 2. **Expense Breakdown** (Top Right)
- Pie chart showing cost distribution
- Categories: Salaries, Marketing, Infrastructure, Office, Operations
- **Insight**: AI identifies largest cost centers

#### 3. **Budget vs Actual Variance** (Middle Left)
- Compares budgeted vs actual spending
- Sorted by variance percentage
- **Insight**: AI highlights biggest discrepancies

#### 4. **Burn Rate & Runway** (Middle Right)
- Dual-axis chart
- Left: Monthly burn rate (how fast cash is spent)
- Right: Runway in months (how long cash will last)
- **Insight**: AI assesses financial sustainability

#### 5. **12-Month Revenue Trend** (Bottom Left)
- Shows revenue growth trajectory
- 1.8% monthly growth assumption
- **Insight**: AI explains growth pattern

#### 6. **Risk Score Trend** (Bottom Middle)
- Tracks financial risk over 12 months
- Green (0-40) = Low Risk
- Yellow (40-70) = Medium Risk
- Red (70+) = Critical Risk
- **Insight**: AI explains risk drivers

#### 7. **Loan Readiness Score** (Bottom Right)
- Funding readiness score (0-100)
- Based on profit margin + runway
- **Insight**: AI assesses funding likelihood

### AI Insights Tab

Generated by **Amazon Nova 2 Lite**:

#### Key Financial Insights
- 3+ top observations about financial health
- Data-driven analysis based on your numbers

#### Financial Risks
- Severity badges (LOW, MEDIUM, HIGH)
- Risk description and context
- Color-coded by severity

#### Strategic Recommendations
- 3+ actionable recommendations
- Prioritized by impact
- Executable action items

## Sample Data

If your financial report has limited data, the dashboard uses intelligent defaults:
- **Revenue**: $100,000 (if not detected)
- **Expenses**: $80,000 (if not detected)
- **Growth**: 1.8% monthly
- **Composition**: Realistic industry ratios

This allows you to see chart functionality even with incomplete data.

## Real Data

## Troubleshooting

### Charts Are Empty
- **Issue**: Revenue/Expenses showing as 0
- **Solution**: Dashboard uses sample data when not detected. Upload a complete financial document.

### AI Insights Not Showing
- **Issue**: "AI insights could not be generated" message
- **Solution**: Check backend logs. Nova API may be slow or timeouts. Refresh page and try again.

### Incorrect Numbers
- **Issue**: Charts showing different values than your document
- **Solution**: This is expected! AI extraction may have small variations. Cross-check with your original document.

## Backend Endpoints

### Get Chart Data
```
GET /api/dashboard/charts/{analysis_id}
Authorization: Bearer {token}
```
Returns all 7 chart datasets in JSON format

### Generate Insights
```
POST /api/dashboard/insights
Authorization: Bearer {token}
Content-Type: application/json

{
  "analysis_id": "...",
  "chart_data": {...}
}
```
Returns AI-generated insights
